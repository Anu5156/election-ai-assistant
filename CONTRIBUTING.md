# Contributing to CivicGuide AI

Thank you for your interest in improving CivicGuide AI! We welcome contributions to make the election process more accessible and transparent.

## How to Contribute
1. **Report Bugs:** Open a GitHub Issue with clear steps to reproduce.
2. **Suggest Features:** Submit an issue with the "feature request" tag.
3. **Pull Requests:** 
   - Fork the repository.
   - Create a feature branch (`git checkout -b feature/AmazingFeature`).
   - Ensure all tests pass (`pytest`).
   - Submit a Pull Request.

## Coding Standards
- Follow PEP 8 guidelines.
- Use type hints for all function signatures.
- Document every module and function with Google-style docstrings.
- Add unit tests for every new feature.
